PromptLabel v0.1.0-beta.1
==========================

This is a beta Windows portable package for PromptLabel.

Before first launch:

1. Download the SAM3 checkpoint file named sam3.pt.
2. Create a models folder beside PromptLabel.exe if it does not exist.
3. Put the checkpoint at:

   models\sam3.pt

The checkpoint is not included in this release package because it is several GB.

Official download:
https://huggingface.co/facebook/sam3/tree/main

Official source:
https://github.com/facebookresearch/sam3

Backup mirror:
https://pan.baidu.com/s/1B1wqcEgTeTckvOlZyVkm3w
Code: 6666

The SAM3 checkpoint is distributed under Meta's SAM License. See
SAM_LICENSE.txt before using or redistributing the checkpoint.

If models\sam3.pt is missing, the main UI can still open and manual annotation
features remain usable, but SAM-assisted annotation will not be available.
